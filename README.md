An offline educational chatbot built with Node.js, Express, and frontend technologies (HTML, CSS, JavaScript).  It answers engineering syllabus-related questions using predefined rules and prompt engineering logic.
# Educational Chatbot

## Overview
This is an offline educational chatbot that answers engineering syllabus-related questions. Built using Node.js, Express, and frontend technologies.

## Features
- Sidebar with FAQ buttons
- Branch selection buttons
- Typing indicator
- Clear chat button

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `node server.js`
4. Open `http://localhost:3000` in browser
